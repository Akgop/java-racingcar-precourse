# 미션 - 자동차 경주 게임

## 구현할 기능 목록

- 사용자로부터 입력 받는 기능
    - 경주 할 자동차 이름
        - 자동차 이름이 1자 이상, 5자 이하인지 검사 및 예외처리
    - 시도할 회수
        - 입력이 올바르게 숫자로 들어오는지 검사 및 예외처리
- 결과를 출력하는 기능
    - 각 차수(회수)별 실행 결과
    - 우승자 안내 문구
    - 예외 상황시 에러 문구
- 자동차 객체관련 기능
    - 사용자로부터 입력받은 이름을 기반으로 자동차 객체 생성
    - 자동차 전진하는 조건 계산
        - 0~9 사이 랜덤한 숫자 생성하고 4이상인지 검사
    - 자동차 전진하는 기능
- 우승자를 계산하는 기능
    - 매 차수(회수)마다 최댓값을 갱신한다.
